\# Rule-Based AI Chatbot 🤖



\## DecodeLabs Internship - Project 1



This project is a simple rule-based chatbot developed using Python.



\### Features

\- Handles greetings (Hi, Hello, Hey)

\- Responds to predefined questions

\- Uses if-else decision-making logic

\- Runs continuously using a loop

\- Supports exit commands (Bye, Exit, Quit)



\### Technologies Used

\- Python 3



\### Sample Conversation



You: hello

Bot: Hello! Nice to meet you!



You: what is ai

Bot: AI stands for Artificial Intelligence.



You: bye

Bot: Goodbye! Have a great day!
