# AI Project 2: Data Classification Using AI

## Objective
Build a supervised learning model to classify iris flowers using machine learning.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Matplotlib
- Visual Studio Code

## Algorithm
K-Nearest Neighbors (KNN)

## Dataset
Iris Dataset (from Scikit-learn)

## Steps
1. Loaded the dataset.
2. Split data into training and testing sets.
3. Trained the KNN model.
4. Made predictions.
5. Evaluated accuracy.

## Result
The model achieved high accuracy on the test data.
